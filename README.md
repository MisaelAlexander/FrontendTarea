Descripción del Proyecto
Techno Meraki es una tienda en línea de electrodomésticos que permite a los usuarios explorar productos, realizar compras y gestionar sus pedidos.
El sistema también contempla un apartado de administrador, encargado de gestionar productos y pedidos.
